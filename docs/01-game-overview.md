# 게임 개요 (Game Overview)

## 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | Instance Growth RPG |
| 플랫폼 | Roblox |
| 장르 | 인스턴스 던전 성장형 RPG |
| 최대 파티 | 4명 |
| 난이도 | Normal / Hard / Extreme / Mythic |

## 게임 컨셉

플레이어는 클래스를 선택하고 파티를 구성해 인스턴스 던전에 입장합니다.
던전을 클리어하면 장비 아이템을 획득하고, 강화/성장을 통해 더 높은 난이도에 도전합니다.

## 차별점

1. **인스턴스 기반** - 파티 전용 던전 공간, 다른 파티와 충돌 없음
2. **역할 기반 전투** - 탱커/딜러/힐러의 역할 분담이 실제 전략에 영향
3. **데이터 드리븐** - 코드 수정 없이 새 클래스/던전/아이템 추가 가능
4. **확장성** - Phase별 컨텐츠 추가 계획으로 장기 서비스 가능

## 플레이어 경험 목표

- 처음 10분: 클래스를 선택하고 기본 전투를 익힌다
- 처음 1시간: 파티를 구성해 던전을 클리어하는 재미를 느낀다
- 장기: 장비 강화와 고난이도 도전으로 성장감을 유지한다
